##  Introdução

O frontend do sistema **Reino Tropical** corresponde à interface web da aplicação, sendo responsável pela interação direta com os usuários.

A aplicação foi desenvolvida com foco em **responsividade**, permitindo acesso tanto em dispositivos móveis quanto em desktops, garantindo uma experiência intuitiva e acessível.

---

##  Funcionalidades

-  Visualização de eventos  
-  Exibição de avisos e conteúdos  
-  Navegação entre páginas da igreja  
-  Interface amigável e responsiva  

---

##  Tecnologias Utilizadas

###  Frontend
- **[React](https://react.dev/)** — Biblioteca para construção da interface  
- **[Vite](https://vitejs.dev/)** — Bundler rápido para desenvolvimento e build  
- **[Tailwind CSS v4](https://tailwindcss.com/)** — Estilização moderna e responsiva  

###  Backend
- **[Node.js](https://nodejs.org/)** — Ambiente de execução JavaScript  
- **[Express](https://expressjs.com/)** — Framework para API REST  
- **[pg](https://node-postgres.com/)** — Integração com PostgreSQL  

###  Infraestrutura & Dados
- **[PostgreSQL 15](https://www.postgresql.org/)** — Banco de dados relacional  
- **[Docker](https://www.docker.com/)** & **Docker Compose** — Conteinerização dos serviços

##  Estrutura do Projeto (Monorepo)

```bash
reino-tropical-frontend/
├── src/                  # Código fonte (React + Tailwind)
├── public/               # Arquivos públicos
├── assets/               # Imagens e mídias
├── Dockerfile            # Configuração da imagem
├── vite.config.js        # Configuração do Vite
├── package.json          # Dependências e scripts
├── eslint.config.js      # Padronização de código
├── .gitignore            # Arquivos ignorados pelo Git
└── README.md             # Documentação do projeto
```
